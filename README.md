# Weather Diary (Дневник погоды)

**Автор:** Качанов Михаил

## Описание
Это графическое приложение для ведения дневника погоды. Вы можете добавлять записи, фильтровать их по дате и температуре, сохранять данные в файл и загружать их при запуске.

## Как запустить
1. Убедитесь, что у вас установлен Python 3.
2. Скачайте проект и перейдите в папку.
3. Запустите команду:
```bash
python weather_diary.py
